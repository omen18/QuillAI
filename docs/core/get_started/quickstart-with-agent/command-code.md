# Install Quill with Command Code

Command Code agent.

## Prerequisites

- [Command Code](https://commandcode.ai) installed and authenticated.

## Install Quill skills

```bash
npx skills add omen18/Quill --agent command-code
```

## Run onboarding

```bash
cmd
```

Then ask:

```text
Use the /quill skill to install and set up Quill.
```

The skill walks the agent through environment checks, profile creation, project scaffolding, and a first query.
