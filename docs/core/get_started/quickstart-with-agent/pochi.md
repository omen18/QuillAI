# Install Quill with Pochi

Pochi coding agent.

## Prerequisites

- [Pochi](https://github.com/vercel-labs/skills) installed and authenticated.

## Install Quill skills

```bash
npx skills add omen18/QuillAI --agent pochi
```

## Run onboarding

Open your project folder in Pochi, then start a new chat session.

Then ask:

```text
Use the /quill skill to install and set up Quill.
```

The skill walks the agent through environment checks, profile creation, project scaffolding, and a first query.
