# Install Quill with Cursor

The Cursor IDE.

## Prerequisites

- [Cursor](https://cursor.com) installed and authenticated.

## Install Quill skills

```bash
npx skills add omen18/Quill --agent cursor
```

## Run onboarding

Open your project folder in Cursor (`File → Open Folder...` or `⌘O` / `Ctrl+O`), then open the chat panel with `⌘L` / `Ctrl+L`.

Then ask:

```text
Use the /quill skill to install and set up Quill.
```

The skill walks the agent through environment checks, profile creation, project scaffolding, and a first query.
